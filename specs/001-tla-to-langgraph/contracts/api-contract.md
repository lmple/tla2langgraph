# API Contract: tla2langgraph Web Server

**Branch**: `001-tla-to-langgraph` | **Date**: 2026-03-25

The local web server exposes a minimal HTTP API on `http://127.0.0.1:{PORT}`.
All endpoints are read-only (no mutation). The server is stateless between requests;
the parsed `StateMachine` is held in application memory for the lifetime of the process.

---

## `GET /`

Serves the single-page UI HTML file.

**Response**: `200 OK`, `Content-Type: text/html`

---

## `GET /api/graph`

Returns the parsed state machine as a JSON graph, ready for Cytoscape.js consumption.

**Response**: `200 OK`, `Content-Type: application/json`

```json
{
  "module_name": "TrafficLight",
  "nodes": [
    {
      "id": "GoGreen",
      "label": "GoGreen",
      "is_initial": true,
      "tla_source": "GoGreen",
      "source_line": 12
    },
    {
      "id": "GoYellow",
      "label": "GoYellow",
      "is_initial": false,
      "tla_source": "GoYellow",
      "source_line": 17
    }
  ],
  "edges": [
    {
      "source_id": "GoGreen",
      "target_id": "GoYellow",
      "label": "light=green",
      "variable": "light",
      "value": "green"
    }
  ],
  "initial_node_id": "GoGreen"
}
```

**Error response** (if parse failed at startup — should not occur since server only starts
on successful parse, but included for robustness):

```json
{
  "error": "parse error at traffic_light.tla:10:5: unexpected token"
}
```

`HTTP 500` with the above body.

## `GET /api/export/skeleton`

Returns the generated LangGraph Python skeleton as a file download.

**Response**: `200 OK`

```
Content-Type: application/octet-stream
Content-Disposition: attachment; filename="traffic_light_graph.py"
```

Body: the generated `.py` file as plain text.

**Example generated content:**

```python
# Generated by tla2langgraph from TrafficLight.tla
# DO NOT EDIT — regenerate from the TLA+ spec

from typing import Any
from typing_extensions import TypedDict
from langgraph.graph import StateGraph, END


class State(TypedDict):
    light: Any  # TLA+ variable: light


def go_green(state: State) -> State:
    # TLA+ action: GoGreen (TrafficLight.tla:12)
    # TODO: implement business logic
    return state


def go_yellow(state: State) -> State:
    # TLA+ action: GoYellow (TrafficLight.tla:17)
    # TODO: implement business logic
    return state


# --- Graph assembly ---

graph = StateGraph(State)
graph.add_node("GoGreen", go_green)
graph.add_node("GoYellow", go_yellow)

graph.add_edge("GoGreen", "GoYellow")  # light=green

graph.set_entry_point("GoGreen")

compiled = graph.compile()
```

## `GET /api/health`

Liveness check for tests and scripting (e.g., wait for server ready).

**Response**: `200 OK`, `Content-Type: application/json`

```json
{"status": "ok"}
```

## Error Handling Summary

| Scenario | HTTP Status | Body |
|----------|-------------|------|
| Successful response | 200 | See above |
| Unknown route | 404 | `{"detail": "Not Found"}` (FastAPI default) |
| Server-side exception | 500 | `{"error": "<message>"}` |
